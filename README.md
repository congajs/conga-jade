conga-jade
==========

Overview
--------

This is a bundle for the [Conga.js](https://github.com/congajs/conga) framework which 
integrates the [Jade Template Engine](https://github.com/visionmedia/jade) into a project.

Configuration
-------------

Example:

    // config.yml
    jade:

        options:
            self: false
            locals: 
            debug: true
            compileDebug: true
            pretty: true
